1. What is the difference between an inline element and a block element?

An inline element stays on the same line and takes up only the space necessary to exist. A block element starts a new line and takes up full available width. 

2. What happens when an element is positioned absolutely? 
    
An element that is positioned absolutely is positioned nearest its positioned ancestors. Sort of overlapping in style.
